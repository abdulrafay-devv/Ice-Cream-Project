# 🍦 Ice Cream Landing Page

A responsive **Ice Cream Landing Page** built using **HTML5** and **CSS3**. This project recreates a modern, colorful landing page for an ice cream brand with a clean layout and attractive UI.

---

## 🚀 Features

* Modern landing page design
* Responsive navigation bar
* Hero section with call-to-action buttons
* Product cards section
* Custom Google Fonts
* Clean and organized HTML & CSS
* Beginner-friendly project structure

---

## 🛠️ Built With

* HTML5
* CSS3
* Google Fonts
* Flexbox

## 💻 Getting Started

### Clone the repository

```bash
git clone https://github.com/your-username/icecream-landing-page.git
```

### Open the project

Navigate to the project folder and open `index.html` in your browser.

No additional dependencies or installation are required.
